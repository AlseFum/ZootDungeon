custom_zh.lang 是示例，请将其放入对应平台的用户数据目录下的 lang 文件夹中：

- Windows: %APPDATA%\.zootdungeon\lang
  - 一般为 C:\Users\<你的用户名>\AppData\Roaming\.zootdungeon\lang
- Linux: $XDG_DATA_HOME/.zootdungeon/lang
  - 若未设置 XDG_DATA_HOME，则为 ~/.local/share/.zootdungeon/lang
- macOS: ~/Library/Application Support/.zootdungeon/lang

如果上述路径不存在，请手动创建相应的 lang 目录后再放入 .lang 文件。

添加之后，在设置的语言界面使用。
