# 伤害/效果范围提示（选格时显示受影响格子）

选格（如放置陷阱、释放范围技能）时，可以让玩家在**鼠标悬停**时看到「若放在这里会影响到哪些格子」，方便合理放置或释放。

## 实现方式

使用 **`GameScene.selectCellWithView(Listener, Select)`** 或 **`selectCellWithView(Listener, Select, Object variable)`**：选格回调 + 范围预览（`CellSelector.Select`）。悬停时调用 `Select.getAffectedCells(hoverCell, variable)`，将返回的格子用半透明红色高亮。

- **无范围预览**：`GameScene.selectCell(listener)`（等价于 `selectCellWithView(listener, null)`）。
- **有范围预览、无变量**：`selectCellWithView(listener, (hover, v) -> ...)`，变量传 null。
- **有范围预览、带变量**：传入第三个参数（如当前物品、半径），同一 Select 可根据变量改变范围，例如炸弹投掷：
  ```java
  GameScene.selectCellWithView(thrower, (hoverCell, item) -> ((Bomb) item).getExplosionCellsForPreview(hoverCell), this);
  ```

## 示例：半径 N 的圆形范围

以「以悬停格为中心、半径 2」为例（类似部分陷阱/技能 AOE）：

```java
import com.watabou.utils.PathFinder;
import com.watabou.utils.BArray;

// 在 getAffectedCells(int hoverCell) 里：
PathFinder.buildDistanceMap(hoverCell, BArray.not(Dungeon.level.solid, null), 2);
ArrayList<Integer> list = new ArrayList<>();
for (int i = 0; i < PathFinder.distance.length; i++) {
    if (PathFinder.distance[i] != Integer.MAX_VALUE) {
        list.add(i);
    }
}
return list;
```

可根据需要把 `2` 换成技能/陷阱的半径，或改用锥形（`ConeAOE`）、直线（`Ballistica`）等逻辑，只要返回受影响的格子集合即可。

## 示例：仅中心格（单格技能/陷阱）

```java
GameScene.selectCellWithView(listener, hoverCell -> java.util.Collections.singletonList(hoverCell));
```

## 相关类

| 类 / 接口 | 说明 |
|-----------|------|
| `CellSelector.Select` | 函数式接口：`getAffectedCells(int hoverCell, Object variable)` 返回受影响格子集合，variable 可为 null |
| `GameScene.selectCellWithView(Listener, Select)` | 选格并附带范围预览（variable 传 null） |
| `GameScene.selectCellWithView(Listener, Select, Object variable)` | 选格并附带范围预览，每次计算时把 variable 传给 Select |
| `GameScene.selectCell(Listener)` | 选格且无范围预览 |
| `AffectedCellOverlay` | 在场景上绘制半透明高亮，由 `CellSelector` 根据 `Select` 更新 |

高亮颜色默认半透明红；若后续需要区分「伤害范围」「治疗范围」等，可扩展 `AffectedCellOverlay.setCells(cells, color)` 的调用处传入不同颜色。
